# Gloss

Gloss is a local-first CLI/TUI command glossary and alias helper for zsh users.

Planned MVP:
- command glossary with descriptions and tags
- browse/search/filter
- add/edit/delete entries
- scan zsh files for aliases/functions
- import suggestions
- manage aliases and sync them safely to ~/.zshrc

Tech direction:
- Go
- Bubble Tea
- Bubbles
- Lip Gloss
- SQLite
